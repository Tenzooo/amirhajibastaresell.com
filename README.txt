How to Upload This Site to GitHub Pages:

1. Go to GitHub and create a new repository.
2. Name it whatever you like.
3. Upload all the files from this folder (index.html and funny.jpg).
   - On GitHub, go to your repository > "Add file" > "Upload files".
4. Commit the changes.
5. Go to the repository Settings > Pages.
6. Under "Build and deployment," select "Deploy from a branch".
7. Select branch "main" (or "master") and folder "/" (root).
8. Click "Save".
9. After a few minutes, GitHub will give you a URL where your site is live.
10. Open the URL in your browser to see your fullscreen image.
